## Requirements

- `bash`

## Installation

Clone the repository:

```bash
git clone <REPO_URL> ~/.dotfiles-rahisi
```

Clone **a dotfiles repository** to `~/.dotfiles`:

```bash
git clone <REPO_URL> ~/.dotfiles
```

Run `provision.sh`:

```bash
bash ~/.dotfiles-rahisi/provision.sh
```
